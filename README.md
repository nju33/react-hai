Hai's React component

## Install

```bash
yarn add react-hai
npm i -S react-hai
```

## Usage

```js
import Hai from 'react-hai';

// ...
  render() {
    return (
      // ...
      <Hai .../>
      // ...
    )
  }
// ...
```
